# SkyGeni – Sales Intelligence Challenge

This repository contains my solution for the SkyGeni Sales Intelligence Challenge:
- Exploratory data analysis (EDA)
- Win rate driver analysis
- Simple decision engine
- Mini system design and reflection

## Structure

- `notebooks/eda_and_model.ipynb` – EDA, modeling, and insights
- `data/skygeni_sales_data.csv` – provided sales dataset
- `src/` – helper scripts (if needed)

## How to run

1. Create and activate a Python environment.
2. Install dependencies (to be added later).
3. Launch Jupyter Notebook and open `notebooks/eda_and_model.ipynb`.
